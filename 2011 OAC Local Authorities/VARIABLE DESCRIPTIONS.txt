### 2011 OAC LOOKUP VARIABLE DESCRIPTIONS ###
#############################################

The following list provides a brief summary of the variables provided in the 2011 OAC Lookups. 

Note: Some of variables report squared Euclidean distance (SED) values. The squared Euclidean distance is the dissimilarity measure used as part of the k-means clustering algorithm that created the 2011 OAC. The larger the value reported, the more dissimilar that Output Area (OA) or Small Area (SA) is to that particular Supergroup/Group/Subgroup cluster centroid.

LOCAL_AUTHORITY_NAME	The name of the Local Authority or equivalent
